# tmp

Répertoire temporaire du repo.

Usage prévu :
- y déposer des fichiers patch générés ponctuellement
- les appliquer en ligne de commande avec `git apply`
- les conserver le temps nécessaire pour téléchargement, test ou application locale

Convention :
- contenu non canonique
- pas destiné à structurer le repo
- peut être vidé ou remplacé à tout moment
